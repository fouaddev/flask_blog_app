About Fouad Boukredine

I love programming in Python! It's a lot of fun!

Hello, my name is Fouad Boukredine, and my final project for the cs521 class is a full stack web application build following modern web development best practices.

This project is a fullstack web application, a simple blog app that allows you to view posts in a modern fashion with cool design. Users can create a new post and publish it in the blog. All posts are displayed in the home page of the blog.

For the front-end I used HTML5 and CSS3, Bootstrap, and the Jinja2 templating engine for Python. Jinja2's performance is impressively fast! it allows to use logic right inside HTML pages. It also allows to reuse code in other HTML pages, and that made my development time faster and fun!

For the back-end, I used the Flask framework that Python offers. I chose it among the other frameworks for its simplicity and agility. It comes with "batteries included" as it has a lot of useful plugins and libraries, which makes developing a web application a lot faster and easier.

As for database, I used SQLite database for its simplicity of use and effectiveness for small projects. It is the right fit for my simple fullstack blog app. I used SQLite with SQLAlchemy which is ORM (object relational mapper) that allowed me to interact with my Python objects easily. For that, I used a Python extension called Flask-SQAlchemy.

Finally, for testing and exception handling, I took care of 404 errors (or "page not found" error), and HTTP500, and also custom typical "try/except" erro handling. The Python framework Flask actually comes with a nice built-in tool to make it a little bit easier to handle errors in a Flask web based application.

I hope you like this blog, and thank you for reading my about me page :)
